# Infosys-General-Election-PowerBI-Dashboard
India General Election Analytics – Power BI Project
📌 Project Description

This project presents an interactive Power BI dashboard developed to analyze and visualize insights from the India General Election dataset. The aim of this project is to transform raw election data into meaningful insights using data analytics and visualization techniques.

The dashboard enables users to explore trends, compare performance across regions, and gain a deeper understanding of election outcomes through an intuitive and interactive interface.


*To analyze election data using interactive dashboards
*To identify trends across states, regions, and constituencies
*To evaluate performance using key analytical metrics
*To understand patterns over time using cohort-style analysis
*To present insights in a clear and visually appealing manner
📊 Dashboard Overview

The Power BI report is divided into two main sections (tabs):

🔹 1. General Election Analytics

This section provides a high-level summary of election data using important KPIs and visualizations.

✅ Key KPIs:
*Total Seats Contested
*Total Seats Won
*Vote Share Distribution
*Top Performing Parties/Regions
*Bottom Performing Parties/Regions
📈 Insights Provided:
*Comparison of party performance across different regions
*Identification of leading and lagging states/constituencies
*Overall election trends and distribution patterns
🔹 2. Cohort Analysis

This section focuses on analyzing patterns over time, similar to cohort analysis.

📌 Features:
Grouping of data based on election phases, years, or regions
Visualization of performance trends over time
Comparative analysis across different election cycles or segments
📊 Benefits:
Helps track changes in performance over time
Identifies consistent strong or weak regions
Supports deeper analytical understanding beyond static KPIs
